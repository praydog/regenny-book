# Types

These are the types that make up the sdkgenny type model. All types inherit from [Object](types/object.md).

The inheritance hierarchy is:

- [Object](types/object.md)
  - [Typename](types/typename.md)
    - [Type](types/type.md)
      - [GenericType](types/generictype.md)
      - [Struct](types/struct.md)
        - [Class](types/class.md)
      - [Enum](types/enum.md)
        - [EnumClass](types/enumclass.md)
      - [Reference](types/reference.md)
        - [Pointer](types/pointer.md)
      - [Array](types/array.md)
    - [Namespace](types/namespace.md)
  - [Variable](types/variable.md)
  - [Function](types/function.md)
    - [VirtualFunction](types/virtualfunction.md)
    - [StaticFunction](types/staticfunction.md)
  - [Parameter](types/parameter.md)
  - [Constant](types/constant.md)
